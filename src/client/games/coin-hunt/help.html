Coin Hunt

Description

Collect all the coins by placing down the roads to create a path accordingly.

Rules

  • Place road tiles next to each other to create a path.
  • Character will try to follow the placed tiles that's a valid path.
  • Collect all the coins by creating a path passing through them.
  • Made a mistake? You can remove the roads with a 3x3 bomb tiles.
  • Be careful with the bombs, though. It can kill the character and reset its position.

Controls

  • Arrow Keys/WASD: Move cursor
  • Space/Left Click: Place road tile
  • X/Right Click: Place bomb tiles
